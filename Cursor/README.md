
# 🚀 Cursor AI Clone (CLI Version)

An AI-powered terminal assistant that generates websites and executes commands automatically using Gemini API.

---

## 🔥 Features

* 🤖 AI Agent with memory
* ⚙️ Function Calling (Gemini API)
* 💻 Executes terminal commands automatically
* 🛡️ Safety filter for dangerous commands
* 🌐 Generates website structure

---

## 🧠 How it works

1. User gives input
2. AI analyzes request
3. AI generates terminal commands
4. Commands executed automatically
5. Output returned

---

## 📦 Tech Stack

* Node.js
* Gemini API (@google/genai)
* readline-sync
* child_process

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/GenAi.git
cd Cursor
npm install
```

---

## 🔑 Setup

Create `.env` file:

```
GEMINI_API_KEY=your_api_key_here
```

---

## ▶️ Run

```bash
node index.js
```

---

## 💡 Example

```
create a portfolio website
```

---

## ⚠️ Note

* Dangerous commands are blocked
* Requires API key
